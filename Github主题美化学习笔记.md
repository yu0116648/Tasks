# 学习笔记

## **参考链接**1：[https://github.com/MaggieAppleton](https://github.com/MaggieAppleton)  

### 亮点介绍：

* 使用**手绘风格**图标，并使用相对**柔和**的配色，让人眼前一新。
* 并不是单一的技术内容，除了技术内容以外，还添加了**创意模块**，适合喜欢创意的人

## **参考链接2**：[https://github.com/taniarascia](https://github.com/taniarascia)  

### 亮点介绍：

* 他使用**模块化**的思想，将内容分为 About、Projects、Writting、Contact四个板块，访客可根据模块化分区快速获取信息。
* 风格**简洁**，没有很多的动态组件，使用少量的文字和徽章，比较适合喜欢**极简风格**和**干净**的访客。
