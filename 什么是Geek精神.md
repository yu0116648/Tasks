# 什么是Geek精神

极客精神可以概括为“**好奇之心，改变之力**”

具体可以理解为：

* 对事物有极强的**好奇心**，并且对**技术**和**创新**有极强的追求，乐于进行技术**钻研**。

* 会**解决问题**，也会**创造工具**，并将自己的知识和成果主动和社区成员**分享**。

* 推崇**开源**，相信**集体的力量**能创造伟大。

# Github 是什么

* Github 是一个基于**Git**的代码托管与协作平台，核心功能包括**版本控制、开源协作、项目管理**等。

* Fork是将他人的**仓库**复制到自己的账号中，可以**独立进行修改**但不影响原项目。

* Pull request 是向原项目提交**代码合并请求**，经过审核后纳入**官方版本**。
